PRODUCT REQUIREMENT DOCUMENT

Product Name: Unravel
Platform: Flutter (iOS & Android)
Scope: Frontend UI Only
Goal: Award-quality calm interface that reduces mental tension

Core philosophy:
When a user opens Unravel, their breathing should slow down.

Not because the app tells them to — but because the interface itself is calm.

PHASE 4 – COMMUNITY EXPERIENCE (Social Reflection Feed)
Purpose

The community section of Unravel allows users to share emotional reflections, achievements, and life experiences in a supportive environment.

The design should resemble a minimal Instagram-style feed, but without competitive social signals like follower counts or popularity rankings.

The tone must remain empathetic and safe.

Core Community Features

Users can:

• View posts in a scrolling feed
• Like posts by double tapping
• Comment on posts
• Share posts via link
• Upload their own posts
• View comments under each post

There will be no direct messaging system.

DESIGN PRINCIPLES

The app must

Never overwhelm
Never guilt the user
Never rush interaction

Every screen must feel like a quiet page in a journal.

Here’s the interesting twist about apps like this.

The most powerful feature isn’t charts or breathing exercises.

It’s the feeling that someone built the space carefully.

When software feels calm, the nervous system notices.

That’s design psychology.

And if Unravel gets that right, people won’t just open it.

They’ll return to it when life gets loud.

NOTE: 
Social media usually compresses people into performances. Filters, metrics, followers. It pushes comparison.

But a space designed differently can do the opposite. It can let people unravel instead of perform.


UPDATED CHANGES DO CONSIDER:

Global UI Updates (Applies to Entire App)
Light Mode & Dark Mode

The app must support two complete themes.

Light Mode
Background: warm cream / soft pastel gradients
Cards: off-white
Text: dark charcoal (#2E2E2E)

Dark Mode
Background: deep indigo / midnight gradient
Cards: dark slate with blur
Text: soft white (#F5F5F5)

Accent colors remain:

Sage green
Dusty lavender
Soft coral

Theme switching should be available in Profile → Settings.

Transitions between themes must be smooth animated fades.

Text Visibility Adjustment

Current design uses very light text.

Update rule:

Primary text: high contrast
Secondary text: medium contrast
Hints: soft gray

Example

Primary: #2E2E2E
Secondary: #5F5F5F
Hints: #8A8A8A

The UI must remain calm but readable.

Elastic Scrolling Effect

All scrollable pages must support elastic overscroll behavior.

When the user scrolls past the top:

A subtle stretch effect appears.

After stretching, a refresh doodle animation appears.

Design

Small rotating doodle icon
Minimal line illustration
Snapchat-style refresh behavior

Animation behavior

User pulls down → doodle appears
Release → rotation animation
Page refresh animation plays

This improves tactile feedback and makes the app feel alive.

Flutter approach

Custom scroll physics + refresh indicator with custom animation.
